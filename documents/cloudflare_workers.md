## 3.创建 Workers

- 1.填写 Workers 的 Script 为下方内容【hostname 填写科学上网的域名，比如 workerproxy.v2ray-agent.com 】

```
addEventListener(
  "fetch",event => {
     let url=new URL(event.request.url);
     url.hostname="这里要修改为自己的";
     let request=new Request(url,event.request);
     event. respondWith(
       fetch(request)
     )
  }
)
```
